## ğŸ“¦ Installation (Development Mode)

1. **Clone the repository**
   ```bash
   git clone https://github.com/mjtech46/MERN-chatWeb.git
   cd MERN-chatWeb
   ```

2. **Backend Setup**
   ```bash
   cd server
   npm install
   npm run dev
   ```
   - Server runs on: `http://localhost:5000`

3. **Frontend Setup**
   ```bash
   cd client
   npm install
   npm start
   ```
   - React app runs on: `http://localhost:3000`
